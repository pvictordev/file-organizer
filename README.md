# File Organizer

This is a simple file organizer script. Run it whenever you feel your folder is getting messy.

## Steps to Run

1. Clone the repository with:
   ```bash
   git clone <repo-url>
   ```
2. Open the folder and navigate to the dist folder:
   ```bash
   cd dist
   ```
3. Run the executable:
   ```bash
   ./main /path/to/folder
   ```
